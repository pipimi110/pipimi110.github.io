CVE-2022-22947

Spring Cloud Gateway SpEL表达式注入

在动态添加路由的过程中，某个filter可以对传入进来的值进行SpEL表达式解析，从而造成了远程代码执行漏洞

https://mp.weixin.qq.com/s/w3et7TzqZ4ctyybEWQ82HQ

影响范围：

Spring Cloud Gateway 3.1.x < 3.1.1

Spring Cloud Gateway < 3.0.7
